# Logo File Location

Please place your logo file here with the name: **logo.png**

## File Location:
```
frontend/public/logo.png
```

## Supported Formats:
- PNG (recommended)
- SVG
- JPG/JPEG

## Usage:
The logo will be automatically used in:
- Login page (left panel)
- Header (right side when logged in)
- Browser tab favicon

## Notes:
- The logo should be at least 200x200 pixels for best quality
- For the login page, the logo will be displayed in white (inverted)
- Transparent background is recommended for PNG format
